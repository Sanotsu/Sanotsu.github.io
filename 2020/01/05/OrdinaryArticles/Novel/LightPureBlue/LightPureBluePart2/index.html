    千秋岁
    西风独凉，寻梦也回廊。
    青灯晃，隐半窗，折书虚半掩，往事便思量。
    相言笑，当时只是道寻常。

    同路行渐远，相逢君欲忘。
    夜半残，栀子香，绕指乱神伤，烈酒寸断肠。
    只空想，但是只是道寻常。

    八月三十
    八月三十寒清晨，落人匆匆雨纷纷。
    有闻不言欲困眼，无心可知病缠身。
    俯仰慨叹虚情绪，转侧思虑假精神。
    枯心从来付殷勤，作梦飞蝶还一人。

    第一章

    “如果我现在向你表白说我喜欢你，你愿意回应我吗？”

    “你知道的，”她正眼看着他，庄重严肃，看起来就像一块死了的木头，“我还不想失去你这个朋友。”

    ……

    我们都比较喜欢春天的，大多数时间里，春天都是欣欣向荣，万物复苏，百花盛开。天气也不那么冷，也不热。空气清新，阳光柔和。

    相比于在秋天里，不知不觉就散发出的浓重的悲怆气味，春天里，人们总是向着幸福看的。

    可惜，春季只有那么长，一旦到了暮春，百花开始凋残，空气开始变得烦躁，人们或许变得抑郁，或者伤怀。

    尽管如此，谁不希望一年之始，就有个好运气呢？

    不过对于小 D 来说，他要的并不是个好运气。他一直不相信运气，运气只是留给那些买彩票的，赌博的，犯罪的人，没有人会因为运气过上美好的生活，即便你买彩票中了大奖。

    谁都不行。

    所以他做的一切，都不是靠运气，全是自己的努力，用时间和汗水拼出来的。他所得到的，都是应当得到的。他所失去的，是他认真付出的，他可能会难过，但没什么值得去后悔的。因为他后悔，他们也回不来。

    至少小 D 一直这样想的。

    这次小 D 感觉到一点伤心。

    这些年来，唯独对这个人，他总是小心翼翼，不想走错一步。

    他认为到目前为止，他还没有犯错，大概上。不过大概也就只能到目前为止。

    他还没有消沉，而且没有浪费一点时间。他回到了他的工作室，接着完成他还未完成的工作，正如什么事都没有发生。至少他的工作室的伙伴都这么认为的。

    事实上，他的确还什么都没有做呢。他就向她询问了一句话而已，就一句。一句话而已，有什么关系。可能就像老师上课讲的无数的话中的一句，有什么关系。就像去听演唱会，少听了一首半曲的，有什么关系。说不定这老师讲的课，本来就毫无意义，说不定这歌手本身长得就不拔萃，所以，有什么关系呢？

    人这一辈子，会说的话太多了；已经说过的话，太多了。谁又会去计较一句两句呢？

    小 D 也这样觉得。

    所以他做的一切都是井然有序，有条不紊，还是一本正经，完全心无旁骛。

    可是他始终还是知道，就算他找到一切理由忘记了这句话，但他也忘不了她说那句话的表情，以前看着她的微笑的脸，嬉笑的脸，大笑的脸，流泪的脸，沉思的脸，困惑的脸，哀愁的脸……说不定也忘了许多。

    那张脸的表情，终于还是让他隐隐作痛。

    据说人只要一悲痛，天就会下雨。

    小 D 本来也想淋淋雨，但他没能淋雨。

    难得的，小 D 晚上到点后，在学校里一个人散了散步。

    很久以前他也是一个人晚上在学校里慢步，而不是在田径场和一堆人吵闹着的一起。而是一个人，放松他的大脑，舒缓他的神经。所以说就算他获得了这样的软件开发，那样编程大赛的奖项，依旧从容自若，心态平和，那是他从来都会有的自我调节。

    他很久没有这样做了，夜里一个人散散步或者去个咖啡店喝口茶看看书之类的事了。可能因为时间久了，大概不仅别人习惯了，自己也习惯了，理所应得的这各种荣誉。

    今天，他又重新一个开始在夜下的校园里散步了，因为这次，他没有能力像往常那样自我调节了。可能也算得上难得的一次放松。

    走在校园的路上，看着并不多的人来往，或许有的自习完匆匆往寝室赶，有的也是闲得无聊出来望望天。其中，他特别注意到一男一女，让他企图安抚的心又隐隐反复。

    他并不认识那两个人，只是他看着，那女生微笑着走在前面，偶尔去敲敲旁边的树，偶尔弯腰理理路边的草，也欣喜地拾起地上的落叶。而那男的不紧不慢地跟在后面，似乎什么事都不做，就好像这女的做的一切都与她无关，他悄悄地，就像生怕打扰了她一样。两个人没有肢体接触，没有拉拉扯扯，更没有一句交流，但是小 D 看得出来，他们是那么的和谐，连包围在他们身边的空气都是快乐的。

    小 D 想要哭，但是他没有流泪，因为他觉得他现在还没到流泪的时候。

    小 D 回过头来，羡慕地看了那一对男女，默默地回寝室去了。

    夜未深，无虫鸣，偶有人声，灯火通明。

    第二章

    “你知道吗？”小 B 对 A 君说，“昨天我和一男生讨论了一件很有意思的问题。”

    “什么问题？”在梳头的 A 君问道。

    “你都不问问是那个男的？”小 B 嘟囔着嘴，故意地说。

    “本来我就不会诧异，”A 君梳直了她那一头浓密的长发，“无论谁都会觉得你是个活泼开朗好交流的人。这么多男生都愿意与你交流聊天也是理所当然的事，哪像我就是有些害怕和他们说话。”

    “那些都是和你不熟的人吧，和你熟悉的人你聊得不是挺嗨的吗？”

    “或许因为有些交流，不得不沟通吧。”A 君自语道，“那你们讨论了一个什么样的问题？”

    “哦，他问，我们女生能不能在起床 5 分钟之内走出寝室。”

    “你怎么说？”

    “我说有必要的话，当然是可以离开的。”

    “不过我觉得大多数时间都是没有必要的。”

    “对啊，为什么要匆匆忙忙地离开寝室呢，至少也要穿衣叠被，洗脸漱口，换鞋梳发啊。一般情况下，5 分钟哪够啊。”

    “的确只能在必要的情况下。”A 君轻声说。

    “不过呢，他开玩笑说了，”小 B 自己也跟着笑了，“他说是因为起床后 5 分钟就出门的女生会吓坏路人的。你说他是不是欠揍啊。”小 B 说完又继续地笑起来。

    A 君没有再说话了，她自顾自地收拾起来。

    小 B 却看在眼里：“你要是再有点幽默感就好了，整天脸上不挂点笑容感觉好闷啊。”不过小 B 停顿了一下，“好在我是挺喜欢你脸红时那害羞的情态的，实在太可爱了。”

    A 君听罢真切脸一红：“你说什么呢，动作快点，要去上早自习了。”

    “哟，上早自习哦，周末都还坚持上早自习，真是个好小孩。”小 B 微微笑着。其它的几个女生都还未醒，或许这种凉快轻柔的天气的确应当睡得熟，睡得香，不宜吵醒。

    按常规道理来说，周六的确应该放松一下，晚点起床才是正确的。早早地起，早早地去上早自习，完全不应该嘛。虽然或许有的同学热爱学习，一周七天都是那样安排的从来不应该有什么休息的时间，就像小 D 那种。

    A 君不是那种人，小 B 更不是，女生们都是喜欢成群结队的，走在一起，吃在一起，玩在一起，小 B 只是陪着 A 君去而已。至于为什么要陪她去，小 B 并不是很清楚这个理由，反正也不是一定需要这个理由，小 B 本来也没有去想过找出这个理由。

    她们轻轻地走出了寝室。

    这时间，若是放在周二，或许也是怕迟到的学生匆匆往教室赶的时间，或许通宵上网的人会比这个时间早一个小时就回来了，但可能也有太饥饿了，逗留了一会儿吃了早餐才回去的。工作日，校园超市这个点也开门了，但不过食堂一直维持到 9：45 都有早饭供应。

    当然，现在离 9:45 还早，不然她们也不会是说去上早自习了。所谓早自习，肯定是一天上课前，起床后。当然，如果把早自习时间算到睡觉去，工作日的睡眠质量或许会大大提升。其实只要头天晚上有好好地早睡，也就没有睡眠不足的说法了。

    不过对他来，就算头天晚上再晚睡，第二天的早自习，他也会出现在那同一间教室，同一个座位上。站在二楼转角处，通过教室后门，就能清楚地看到他端坐的背影。当然，你看得久一点，可能看到斜坐的背影，趴在桌上的背影，取了眼镜扑在桌上的背影。只透过后门，当然是看不到他的脸的，当然，他的脸也不是很好看。其实就算是他的背影，也不是很好看。一般情况下，也没有人刻意去看。

    但这时候，小 B 和 A 君就站在二楼转角处。

    “走了吧，差不多也看够了，走了。”小 B 催促着。

    A 君也没有再看了，被小 B 拉着胳膊走开了。

    “你不会真的喜欢他吧？他有哪点值得你喜欢的？”小 B 总是这样疑惑又亲切地问 A 君，一直都这样问。只是 A 君从来没有回答过，偶尔会脸红，偶尔会微笑一下，却没有多余的一个字。

    是的，那男的每一天都会去上早自习，就算是在周末。其实他并不会因为在上早自习就好好学习，有时会看看英语书，有时会写写东西，有时就只是在玩着手机——虽然 A 君没有看到，或许只是因为 A 君每次来的时间都差不多，看的时间也不长，正好错过了。总之，他并不是一个热爱学习的人，他同学知道，他自己也是知道。

    还有，小 D 也知道。

    第三章

    小 D 从来不去教室上早自习，他并不觉得上早自习不应该，或者去教室上早自习不应该。只是他知道他会去上早自习，而且不出意外还是在同一间教室的同一个座位上，所以他不会去教室上早自习。

    就算刻意避开了他上早自习的教室，但还是有可能在去上早自习的路上遇到。

    所以他很早就起床，比他早就起床，走去教室不同路的工作室去。

    小 D 知道这个人是谁，而且非常不喜欢他，特别地厌恶。要问问具体的原因，小 D 不想一一去提起，也不会去提起。

    很多很多。

    他们不是同一个专业的，但有些课却是大家都要学的，他不会跟他选同一个老师的课，不会在同一个时间点去吃饭，不乐意去做相似的事，真是不想和他出现在同一个世界里。

    其实 C 君也是这么想的。

    不过 C 君从来没有表现出来，他不像小 D 那样看着 C 君就十分愤怒，火冒三丈，失去理智，就想上去狠狠一顿打——虽然没这么做，C 君从来不。

    他可能只会默默坐在讲台下某个角落，看着小 D 拿奖，或者倚着某个门后听小 D 参加某个项目答辩，甚至远远地看着小 D 气冲冲地离开。

    其实他们之间并没有什么交集，也不是曾经的朋友，只是他们两个或者都不喜欢对方，而且两个人的表现方式还不同。

    他们之间有说过话吗？或者他们自己也不记得了，可能说过，不过是谁开的口，谁又会愿意听呢？

    那么，他们又是为什么如此了解对方呢？

    一般外人，都很难知道。

    C 君每天都去上早自习，而且他自己也知道自己应该是在上早自习，不过有时候他将上早自习翻译成在某个时间，在某个教室里的某个座位上坐着，这个早自习的意义，在他看来就只是坐在哪儿而已。对于一般人来说，上早自习应该看看英语书，读读诗文，或者做做作业之类的，不然光坐在那里什么都不做，简直是毫无意义，浪费时间，还不如好好在寝室里多睡一会呢。

    不过，C 君将早自习的含义翻译了之后，便已经习惯了，这么个做法，每天早上醒来，去固定的教室坐在固定的座位上持续固定的一段时间。

    有些习惯，一旦养成就很难改掉，不然也不能称之为习惯，习惯也有好习惯，坏习惯。好习惯自然不用改，坏习惯自然应当改，不过难的就是怎么去区分好习惯和坏习惯。

    一个人怎么去评价好习惯或者坏习惯呢？

    是好，还是坏呢？

    不过 C 君自己并不在意，既然已经养成了这个习惯，那就好好地保持下去吧，何必去计较到底是好习惯还是坏习惯呢？这是个伤脑筋的事情。

    C 君从来不喜欢做伤脑筋的事情。

    不过可悲的事实就是，总是有人去做某些伤脑筋的事情，每个人理所当然总会遇到一些，不去做入 C 君似的，那就置之不理，等着他们自行变化。那当然也有积极去做的，非得去做，一定要去做这种事的人，而且很巧的是，这种人一般就会出现在不想去做这种事的人周边。

    G 君就是这种人，而且他就出现在 C 君的周边。

    有时偶仔细想想，说 G 君喜欢做伤脑筋的事，这个论判是不正确的。一般情况下，没人喜欢去做伤脑筋的事情，只是有时候不得不去做而已，不得不做是一种消极的态度，所以说像 G 君一样的人，其实只是想做好伤脑筋的事而已。反正不得不做，做好了，再有也就不算伤脑筋的事了。

    当然，他也不会去计较 C 君每天都上早自习是不是一个好习惯，因为他有些自己的习惯，比如周末或放假就睡过整个上午，吃了午饭就好好去学习。

    一直都是。

    无论是学各种计算机编程语言，还是市场营销管理，或者文学鉴赏分析，亦或马列毛邓理论，总是想找点东西学，而且还想学得适当好。

    因为他是 C 君的朋友，这才应该是他最伤脑筋的事。

    有些渊源说起来就长了，G 君也懒得去提起，好在 C 君经常不去找他，他就可以一如既往自主地安排。

    “hi，G。”

    “Oh，come on，not today！”G 君一脸无奈的样子。

    “别啊，我还什么都没有说呢。”本来 C 君不想说这句话的，但难得心情不错，便附和了一下。

    “又要帮你弄什么？”G 君已然发现话说得越多，越浪费时间，还不如直接问来得快。

    “没什么，我就过来看看你，打声招呼。”C 君说着。但其实呢，他的确找 G 君有事的，但突然没了兴致。

    “不要这么说，我突然感到后背一阵凉意。”G 君越发觉得今天的事可能有些棘手。

    “好了，注意点身体，别坏掉了。”C 君再次扫了一眼，便真的离开了 G 君的工作室。

    这让 G 君始料未及，C 君走后好一会，G 君都还在隐隐微笑。

    G 君自然明白 C 君为什么匆匆离开，啥事没做，要么看到小 D，要么就是看到她了，尽管之前从没看到过。

    第四章

    E 女并没有反驳什么，她只是站了起来，径直走向小 D，让后给了他一个拥抱：“我第一次和 C 君交流时，他就是这样给了我一个拥抱，只是一个单纯的拥抱。我并不知道他当时是为了安慰我，还是在同情我，不过今天，我告诉你，我原谅你了。”

    小 D 听了这话，也终于放下心来，双手象征性地拍拍她的后背。

    （原结尾）

    这些事情当然还有后续，A 君之后无论看到小 D 或者 C 君，都会有更多的笑容，说话也更加干脆利索，偶尔也会开开小玩笑。小 D 虽然偶尔会和 E 女讨论一下学习工作，但似乎仅限于此。E 女看到 C 君后也会难得地笑一笑，打声招呼。C 君也会继续写诗文，偶尔还和 A 君唱和，当然也不忘开始认真学习功课，只是 C 君和小 D 依旧不愿意看到对方，即使意外相逢，依旧还是怒目而视，不言一语，快速离开，一切好似如初。但大家比以前心情更加愉悦。

    当情感线一直是线性且没有支路的时候，每个人都不会太轻松。有的人不愿意轻易放弃，一直固有的坚持。有的人偶尔回头看看，才找到休憩的间歇。A 君有时候问小 D，如果自己当初也像小 D 一样坚持着自己的心意，一直等待着 C 君，小 D 会怎么做。小 D 只是说，他耐得住寂寞，而且肯定比 A 君更能坚持，因为他已经不需要再往别的地方看了。

    C 君偶尔也会去工作室，但不是经常去找 G 君了，可能 G 君一直批评 C 君重色轻友，但依旧毫无办法。还在这会儿偶尔小 H 会打个电话叫他一起出去打打篮球之类的，G 君还算比较满意，反正也没必要再担心什么。

    又是一个周四晚上，E 女并没有再去图书馆，她已经答应了 C 君每周四晚上一起在校园里散散步，抽点时间舒缓压力，也更多地交流交流，分享话题。

    这周四，他们又遇到了那对男女，女生依旧信步走在前方，时而拍拍树，时而拾起树叶，时而摸摸小草。而男的只轻轻地走在后面，就像是两个人走的太近，生怕会打扰到了她。

    C 君和 E 女远远看着他俩的背影，觉得十分轻松和谐，只不过现在 C 君不会再羡慕更或者嫉妒了，他牵着她的手问到：“或许我们以后也会像他们一样。”

    “会的。”E 女笑了，晚风轻拂，树影光浅，一直板着脸的 E 女，她的笑容原来真的那么美。

    只是他们可能不会像那一对平和的男女一样，大地因为 E 女还是没有告诉 C 君为什么当初会说活着对她来说是件痛苦的事，C 君也同样没有解释自己喜欢她的原因。但是，谁也不知道 E 女是否依旧还那样觉得，他们的感情真的需要 C 君说出甜言蜜语。其实有的事不需要太过追究某些点上，和谐平静的生活，稳定美好的关系，对于两个人，已经足够了。

    无论是对于小 D 和 A 君，还是 C 君和 E 女。

    坚持做对的事情，根本没必要畏手畏脚，有些事物与人，从来不会错过。

    （原结尾完）
